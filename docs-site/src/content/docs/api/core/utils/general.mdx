# `codeweaver.core.utils.general`

General utilities for CodeWeaver.

## Function: `deep_merge_dicts`

```python
deep_merge_dicts()
```

Deep merge two dictionaries.

## Function: `dict_set_to_tuple`

```python
dict_set_to_tuple()
```

Convert all sets in a dictionary to tuples.

## Function: `ensure_iterable`

```python
ensure_iterable()
```

Ensure the value is iterable.

Note: If you pass `ensure_iterable` a `Mapping` (like a `dict`), it will yield the keys of the mapping, not its items/values.

## Function: `generate_collection_name`

```python
generate_collection_name()
```

Generate a collection name for the project.

## Function: `rpartial`

```python
rpartial()
```

Return a new function that behaves like func called with the given arguments from the right.

## Function: `supported_language_count`

```python
supported_language_count()
```

Return the count of supported languages.

## Function: `supported_languages`

```python
supported_languages()
```

Return the supported languages.
