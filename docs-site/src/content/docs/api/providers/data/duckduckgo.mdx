# `codeweaver.providers.data.duckduckgo`

DuckDuckGo data provider tool implementation.

## Class: `DdgsResultItem`

A single DuckDuckGo search result item.

## Class: `DdgsSearchResults`

Wrapped results from a DuckDuckGo search, conforming to MCP object schema requirements.

## Class: `DuckDuckGoSearchTool`

A tool for searching DuckDuckGo.

## Function: `duckduckgo_search_tool`

```python
duckduckgo_search_tool()
```

Factory function to create a DuckDuckGoSearchTool instance.
